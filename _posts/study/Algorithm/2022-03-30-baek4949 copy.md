<h2>코드</h2>
```py
#균형잡힌 세상
while(True):
    a=input()
    s=[]
    tmp=True
    if a=='.':
        break
    for i in a:
        if i =='(' or i=='[':
            s.append(i)
        elif i==')':
            if not s or s[-1]=='[':
                tmp=False
                break
            else:
                s.pop()
        elif i==']':
            if not s or s[-1]=='(':
                tmp=False
                break
            else:
                s.pop()
    if tmp==True and len(s)==0:
        print("yes")
    else:
        print("no")

```
